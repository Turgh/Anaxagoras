# 🛰️ Project Anaxagoras

> **"Per Audacia Ad Astra"** > *An edge-AI cognitive vision system inspired by the first philosopher to declare celestial bodies as sacred rocks. Built for orbital debris tracking and lunar regolith analysis.*

## Project Overview
This repository contains a simulated **Edge-AI Space Domain Awareness (SDA)** system. Operating entirely locally, the system simulates a satellite or rover's optical payload tracking orbital debris (Space Junk) and analyzing rock features to avoid collisions and map lunar or planetary environments.

Designed with **NVIDIA Jetson Orin Nano** hardware constraints in mind, it utilizes highly optimized, quantized computer vision models to ensure low latency and high energy efficiency in deep space environments where cloud computing is impossible.

---

## Tech Stack & Architecture
- **Language:** Python 3
- **Core AI Engine:** Ultralytics YOLOv8 / YOLOv11 (Object Detection)
- **Image Processing:** OpenCV (Open Source Computer Vision Library)
- **Target Hardware (Edge):** NVIDIA Jetson Orin Nano (8GB) running localized Docker containers
